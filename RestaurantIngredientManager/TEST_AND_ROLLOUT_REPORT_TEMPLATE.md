# 测试与上线报告模板

## 1. 单元测试与集成测试
- 时间换算：分钟/小时/天/月/年/自定义、边界值、DST 场景。
- 打印状态同步：断线重连、跨页面状态共享、重启恢复配对。
- 模板渲染：字段开关、画布模板编码解析、预览填充一致性。

## 2. 覆盖率报告
- 建议命令：`xcodebuild test -scheme RestaurantIngredientManager -enableCodeCoverage YES`
- 产物：`DerivedData/Logs/Test/*.xcresult`
- 覆盖率门槛：单元+集成总体 >= 90%

## 3. 可用性测试记录
- 任务：新建模板 → 添加图片 → 调整字体 → 打印
- 样本：5 名非技术用户
- 指标：成功率 >= 90%，平均耗时 <= 2 分钟

## 4. 上线后 7 天监控
- 打印状态同步延迟（P95）<= 1s
- 模板编辑错误率 <= 1%
- 断线自动恢复成功率 >= 99%

## 5. 用户操作视频清单
- 视频1：分类配置与动态字段录入
- 视频2：模板画布编辑与图片上传
- 视频3：打印机断线重连与测试打印
